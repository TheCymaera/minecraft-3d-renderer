# Usage
Install the plugin in a void world. The coordinates are hard-coded.

Load a scene:
```
/renderer3d empty
/renderer3d rainbowTriangle
/renderer3d rotatingCube
```

Get items for controlling the camera:
```
/renderer3d getItems
```
